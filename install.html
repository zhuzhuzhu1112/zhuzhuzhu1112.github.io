安装简洁音乐播放器

简洁音乐播放器是一个渐进式Web应用（PWA），您可以将其安装到您的设备上，获得更好的使用体验。

方法一：通过浏览器安装

1. 使用Chrome、Edge或Safari等现代浏览器打开应用

2. 在地址栏右侧或菜单中寻找"安装"或"添加到主屏幕"选项

3. 点击安装按钮并确认

方法二：手动添加到主屏幕

iOS设备：

  1. 使用Safari浏览器打开应用
  2. 点击底部的分享按钮
  3. 选择"添加到主屏幕"
  4. 点击"添加"确认

Android设备：

  1. 使用Chrome浏览器打开应用
  2. 点击菜单按钮（三个点）
  3. 选择"添加到主屏幕"
  4. 点击"添加"确认

安装后的优势

  • 可以从设备主屏幕快速启动
  • 离线支持，无需网络连接也能使用
  • 获得更好的性能和用户体验
  • 与本地应用程序体验一致

注意事项

  • 安装过程完全免费且安全
  • 应用不会占用太多存储空间
  • 您可以随时卸载应用
