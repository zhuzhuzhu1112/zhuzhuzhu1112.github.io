# 简洁音乐播放器

一个基于Web的简洁本地音乐播放器，支持PWA安装到设备上使用。

## 功能特点

- 扫描并播放本地音乐文件
- 支持播放、暂停、上一曲、下一曲等基本控制
- 音量调节和进度条控制
- 响应式设计，适配不同设备
- 支持作为PWA安装到设备上
- 离线使用功能

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
- PWA (Progressive Web App)

## 使用方法

1. 点击「扫描本地音乐」按钮选择音乐文件
2. 使用播放控制按钮控制音乐播放
3. 点击「安装应用」可以将应用安装到设备上
